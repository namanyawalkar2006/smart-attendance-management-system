smart Attendance Management System

A smart, scalable attendance system designed for academic institutions.

## Problem
Manual attendance is time-consuming, error-prone, and lacks analytics.

## Solution
This system provides:
- Subject-wise attendance
- Session-based saving
- Student-wise profiles
- Monthly & yearly analytics
- Risk attendance identification

## Features
- Role-based access (Teacher / Student / HOD)
- Real-time data storage
- Clean and intuitive UI
- Secure authentication
- Scalable architecture

## Tech Stack
- Frontend: Web / Flutter
- Backend: Firebase (Real-time Database)
- Auth: Google / Email
- Hosting: Cloud-based

## Future Scope
- Face recognition attendance
- Parent notifications
- AI-based attendance prediction
